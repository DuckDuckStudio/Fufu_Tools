# 芙芙工具箱<br>

本仓库存放着我的写的一些小工具，可以自行按照说明使用。<br>
**如果您对使用方法有任何疑问，请先阅读小工具目录下的使用说明或者文档**。<br>

## 对于lite版的芙芙工具箱<br>
对于lite版的芙芙工具箱，在使用前，你必须先安装`python`（编写使用的是`python 3.12`）并正确配置！<br>
另，对于代码中使用到的库文件请自行安装！（可以使用`pip`安装，或者运行环境配置文件夹下的`库文件`）<br>
